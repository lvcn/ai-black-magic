## 🚀 Quick Start

**Step 1**: Clone this repo

```bash
git clone https://github.com/lvcn/ai-black-magic.git
```

**Step 2**: Pick a Skill File from the `skills/` directory

**Step 3**: Feed it to your AI assistant

| Tool | How to load |
|------|------------|
| **Cursor** | Copy to `.cursor/rules/` — auto-loaded on every chat |
| **ChatGPT / Claude** | Paste at the beginning of conversation, or upload as file |
| **GitHub Copilot** | Reference the file in Copilot Chat |
| **Windsurf / Cline** | Add to project rules or paste in chat |

That's it. Your AI now operates at expert level.
